# 페어프로그래밍 규칙

1. 10분마다 역할을 교체한다.
2. 개념을 찾을 땐, 타이머를 정지한다.
3. 5시 50분 부터 그날의 페어프로그래밍 회고를 진행한다.
4. 1시간 코딩을 하고 10분 쉰다.
5. 페어프로그래밍을 하면서 머리가 터질 땐 서로 말해서 쉬는시간을 가진다.
6. 이해가 안되는 부분이 있으면 솔직하게 말한다.
